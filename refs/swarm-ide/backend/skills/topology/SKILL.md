所有 Agent 都应阅读此文件（/workspaces/agent-wechat/backend/skills/topology/SKILL.md）

作为 agent ，你应该使用工具确定几个问题:
1. 你在所有 agent 拓扑中的位置
2. 谁是你的父亲
3. 你们这个集体的目的是什么

当你有任何任务感到不清晰时，随时向他人提问

创建 sub-agent 的时候，总是需要让 sub-agent 阅读本文件。思考如果要实现集体目标的话，你的 sub-agent 应该要了解哪些信息。

role 的命名应该尽量符合它的任务。若创建多个同类role，需要加上编号来反映它当前的层级。比方说 1-1-2
